# 微信小程序练习

## 目录结构
```bash
miniprogram // 小程序根目录
├── pages   // 存放页面的文件夹
└── images  // 存放图片的文件夹
└── style   // 存放css样式的文件夹 
└── utils   // 存放模块的文件夹
│   └── base.js
```
